# Public Gameplay Recording Audit

No copyrighted video/audio file is copied into this repository. This document preserves lawful source links and observable high-level mechanics only.

## Sources

- Amazon listing: https://www.amazon.com/Narkuma-the-black-sword/dp/B07B973X4M

- 2021 project/gameplay announcement: https://www.youtube.com/watch?v=4ATQAGBV_Hk
- Archived podcast metadata mentioning chat: https://archive.org/details/xa9pysmmqaqjcxratb49lygvvxo3szhhyfwq2jv0

## Publicly observable mechanics

- Free-form voice commands rather than menu-only yes/no choices.
- Character naming and persistent continuation from the previous stopping point.
- Marketplace starting context.
- Palace conversation, quests, kill objectives, proof/trophy collection, and level advancement.
- Temple prayer and attribute choice.
- Shop `list` command, priced weapon/armor catalog, and item purchase.
- Open-world monster killing, leveling and riddles.
- Amazon description: other heroes, conversation, cooperative play, combat groups, brotherhoods, companions, weapons/equipment, landscapes, caves, cities and player-created maps.
- Podcast metadata confirms that a chat function is explained, but the media itself is access-restricted and its private details are not reproduced here.

## Web implementation equivalents

- Supabase public/private/direct/custom rooms and realtime delivery.
- Manual loot, public location drops, inventories, storage and equipment.
- Turn-based transparent combat with companions and online shared damage.
- Palace quests/advancement, Temple prayer, physical marketplaces and quantity shops.
- Persistent multi-hero cloud saves and voice commands.

Exact proprietary dialogue, secret spells, maps, puzzle solutions and internal combat/chat formulas are intentionally not copied.
