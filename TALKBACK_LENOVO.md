# Lenovo Android Tablet: TalkBack and Chat Voices

1. Open **Settings → Accessibility → TalkBack**.
2. Turn TalkBack on. Confirm with the accessibility shortcut if requested.
3. Open **Settings → System → Languages & input → Text-to-speech output**. On some Lenovo tablets this is under **Accessibility → Text-to-speech output**.
4. Select **Speech Services by Google** as the preferred engine. Install/update it from Google Play if absent.
5. Open the gear beside the engine → **Install voice data**.
6. Choose each required language and download one or more voices. Voice names vary by Android version and installed language pack.
7. Set speech rate and pitch, then use **Play** to test.
8. Open the game in current Chrome and allow Sound and Microphone permissions.
9. In the game open **Settings → Default spoken-chat voice**. The list now uses actual voices reported by Android/Chrome—not Boy Voice 1 codes.
10. Select a voice and press **Test Selected Voice**.
11. Enable **Speak incoming messages**.

TalkBack basics: swipe right/left to move focus, double-tap to activate, and use two fingers to scroll. Internal voice identifiers are never printed in chat messages.
